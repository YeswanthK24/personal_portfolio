# 🌐 3D Developer Portfolio (portfolio_v2)

An interactive **3D portfolio website** built using modern frontend technologies to showcase my projects, skills, and experience in a visually engaging way.

🚀 Available for freelance projects (React / Node.js / UI Fixes)

---

## 🚀 Live Demo
🔗 [https://personal-portfolio-psi-seven-29.vercel.app/](https://personal-portfolio-psi-seven-29.vercel.app/)

---

## 🛠️ Tech Stack

- ⚛️ React.js  
- 🎨 Tailwind CSS  
- 🌌 Three.js / React Three Fiber  
- 🧩 Framer Motion  
- 📦 Vite  

---

## ✨ Features

- 🧑‍💻 Interactive 3D elements and animations  
- 📱 Fully responsive design  
- 🎯 Smooth scrolling and transitions  
- 📂 Projects showcase section  
- 📄 Resume / About section  
- 📬 Contact form integration  
- ⚡ Fast and optimized performance  


## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/YeswanthK24/portfolio_v2.git
cd portfolio_v2
```

### 2. Install dependencies
``` bash
npm install
```
### 3. Run the project
```bash
npm run dev
```
